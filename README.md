# MasteringSoftwareDevelopmentCapstone

This package contains functions to clean and visualise earthquake data.

The following 6 functions are included in this package:
<ol>
  <li>`eq_clean_data()`
  <li>`eq_location_clean()`
  <li>`geom_timeline()`
  <li>`geom_timeline_label()`
  <li>`eq_map()`
  <li>`eq_create_label()`
</ol>

For worked examples, please refer to vignette documention in  [`./inst/doc/MasteringSoftwareDevelopCapstone.html`](https://github.com/elyminator91/MasteringSoftwareDevelopmentCapstone/inst/doc/MasteringSoftwareDevelopmentCapstone.html)
